# Student Management System

A web-based student management system built with HTML, CSS, and JavaScript.

## Features
- Add and manage student records
- View students by section (A, B, C)
- Simple and clean UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Just open Student.html in your browser.
